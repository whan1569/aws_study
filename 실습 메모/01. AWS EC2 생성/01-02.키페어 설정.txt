1. AWS Management Console 접속
   - AWS 콘솔에 로그인 후 EC2 대시보드로 이동

2. 키 페어 생성
   - 왼쪽 메뉴에서 Network & Security > Key Pairs로 이동
   - Create Key Pair를 클릭

3. 키 페어 이름 지정
   - 키 페어 이름을 지정. 예: <project-name>-key
   - 특별한 규칙은 없지만 식별하기 쉬운 이름을 추천

4. 키 페어 형식 선택
   - PEM 형식으로 다운로드하는 것이 일반적 (Windows에서는 PPK 형식으로 변환 필요)

5. 키 페어 다운로드
   - 개인 키 파일을 다운로드 받음
   - 파일을 안전하게 보관, 절대로 공개하지 않음
