1. AWS EC2 (Elastic Compute Cloud):
    - 서버 배포 및 관리: EC2 인스턴스를 사용한 서버 프로비저닝 및 운영 경험.
    - 자동 확장(Scaling): 오토 스케일링 그룹을 설정하여 트래픽 증가에 따라 서버가 자동으로 확장되고 축소되는 설정.
    - 보안 그룹과 네트워크 ACL을 이용한 보안 설정.

2. AWS S3 (Simple Storage Service):
    - 대용량 파일 저장 및 백업.
    - 정적 웹사이트 호스팅: S3를 사용한 정적 웹사이트 배포 경험.
    - 버전 관리, 생성된 파일의 저장 및 정리 정책을 설정하는 경험.

3. AWS RDS (Relational Database Service):
    - MySQL, PostgreSQL 등 RDS 데이터베이스 배포 및 운영 경험.
    - 백업 관리, 자동화된 스냅샷, PITR(Point-in-Time Recovery) 설정 경험.
    - **고가용성(AZ 배포)**와 읽기 전용 복제 설정 경험.
    - 성능 튜닝 및 쿼리 최적화 경험.

4. AWS Lambda:
    - 서버리스 아키텍처: Lambda 함수를 사용하여 서버 없이 코드 실행.
    - API Gateway와 연동하여 RESTful API 구축 경험.
    - 이벤트 기반 아키텍처에서 Lambda를 사용하여 비동기 처리.

5. AWS VPC (Virtual Private Cloud):
    - 네트워크 아키텍처 설계: VPC, 서브넷, 인터넷 게이트웨이, VPN 설정.
    - 프라이빗 서브넷과 퍼블릭 서브넷 구분하여 보안과 성능 최적화 경험.
    - Route 53을 통한 DNS 관리 및 로드 밸런싱 경험.

6. AWS CloudFormation:
    - 인프라 코드화: CloudFormation 템플릿을 사용하여 인프라를 코드로 정의하고 자동화.
    - 스택 관리 및 리소스 자동화 배포 경험.

7. AWS IAM (Identity and Access Management):
    - 권한 관리: IAM 역할, 사용자 및 그룹을 설정하여 보안 및 접근 제어 경험.
    - 정책 작성 및 관리: 세밀한 접근 제어를 위한 IAM 정책 설정 경험.

8. AWS CloudWatch:
    - 모니터링 및 로그 관리: EC2, RDS 등에서 CloudWatch를 통해 성능 모니터링 및 알림 설정.
    - 로그 수집 및 메트릭 모니터링을 통한 성능 최적화 경험.

9. AWS SNS/SQS (Simple Notification Service/Simple Queue Service):
    - 이벤트 기반 시스템 구축: SNS/SQS를 활용한 비동기 메시징 및 알림 시스템 구축.
    - 분산 시스템 설계 및 확장성 고려한 메시징 큐 사용 경험.

10. AWS Elastic Beanstalk:
    - 애플리케이션 배포 자동화: Beanstalk을 이용한 애플리케이션 배포 및 스케일링 관리.
    - CI/CD 파이프라인과의 통합을 통한 자동화 배포 경험.

11. AWS CloudTrail:
    - 로그 추적 및 감사: AWS 환경에서 API 호출을 추적하여 보안 감사 및 컴플라이언스 유지.
    - CloudTrail 로그를 통해 활동 모니터링 및 문제 해결.

12. CI/CD 및 DevOps 도구:
    - AWS CodePipeline, CodeBuild, CodeDeploy 등을 이용한 CI/CD 파이프라인 구축 및 운영 경험.
    - 자동화된 빌드 및 배포 프로세스 설정 경험.

13. AWS 비용 관리:
    - AWS 비용 최적화: 리소스 사용량을 모니터링하고 비용 절감을 위한 설정(예: Reserved Instances, Spot Instances).
    - AWS Cost Explorer와 Budgets를 통한 비용 분석 및 예산 설정.

AWS 실무 경험을 평가하는 기준:
    - AWS 서비스의 전반적인 이해: 각 서비스가 어떤 역할을 하는지, 언제 어떤 서비스를 사용하는 것이 좋은지 이해하고 있는지.
    - 자동화 및 인프라 관리: AWS에서의 리소스 자동화 및 배포 관리 경험.
    - 클라우드 보안: IAM, VPC, 암호화 등 클라우드에서의 보안 관리 경험.
    - 확장성 및 가용성: 고가용성, 분산 아키텍처, 성능 최적화 관련 경험.
    - 문제 해결 능력: AWS에서 발생할 수 있는 문제를 효율적으로 해결하는 경험.
    - 비용 관리: 리소스 최적화 및 비용 관리 경험.
