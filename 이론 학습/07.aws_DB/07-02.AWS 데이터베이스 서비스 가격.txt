AWS 데이터베이스 서비스 가격 개요 및 비용 예시 (100GB, 1시간 기준)

1. Amazon RDS (관계형 데이터베이스 서비스)
   - 지원 DB: MySQL, PostgreSQL, MariaDB, Oracle, SQL Server
   - 요금:
     - 인스턴스 비용: 시간당 $0.032 ~ $3.30 (약 4,200원 ~ 437,000원)
     - 스토리지 비용: SSD 기준 1GB당 $0.115 (약 150원) / 월
     - 백업 비용: 프로비저닝한 스토리지 크기와 동일한 양까지 무료, 초과 시 1GB당 $0.095 (약 120원)
   - 비용 예시:
     - 100GB 데이터베이스를 `db.t3.medium` 인스턴스로 1시간 사용
     - 인스턴스 비용: $0.0464 (약 60원)
     - 스토리지 비용: $0.016 (약 21원)
     - 총 비용: $0.0624 (약 81원)

2. Amazon Aurora (고성능 관계형 데이터베이스)
   - 지원 DB: MySQL, PostgreSQL 호환
   - 요금:
     - 인스턴스 비용: 시간당 $0.10 ~ $2.40 (약 13,500원 ~ 320,000원)
     - 스토리지 비용: 1GB당 $0.10 (약 130원) / 월
     - 백업 비용: 1GB당 $0.021 (약 30원)
   - 비용 예시:
     - 100GB 데이터베이스를 `db.r6g.large` 인스턴스로 1시간 사용
     - 인스턴스 비용: $0.1344 (약 180원)
     - 스토리지 비용: $0.014 (약 18원)
     - 총 비용: $0.1484 (약 198원)

3. Amazon DynamoDB (비관계형 데이터베이스)
   - 요금:
     - 읽기 요청: 초당 단위당 $0.00013 (약 0.017원)
     - 쓰기 요청: 초당 단위당 $0.00065 (약 0.085원)
     - 스토리지 비용: 1GB당 $0.25 (약 340원) / 월
   - 비용 예시:
     - 100GB 데이터베이스, 초당 1,000 읽기 / 500 쓰기 요청 기준
     - 읽기 요청 비용: $0.13 (약 170원)
     - 쓰기 요청 비용: $0.325 (약 430원)
     - 스토리지 비용: $0.35 (약 470원)
     - 총 비용: $0.805 (약 1,070원)

4. Amazon Redshift (데이터 웨어하우징)
   - 요금:
     - 인스턴스 비용: 노드당 시간당 $0.25 ~ $4.20 (약 34,000원 ~ 560,000원)
     - 스토리지 비용: 1GB당 $0.085 (약 115원) / 월
   - 비용 예시:
     - 100GB 데이터베이스를 `dc2.large` 노드로 1시간 사용
     - 인스턴스 비용: $0.25 (약 340원)
     - 스토리지 비용: $0.012 (약 16원)
     - 총 비용: $0.262 (약 356원)

5. Amazon ElastiCache (Redis 기반 인메모리 캐시)
   - 지원 DB: Redis, Memcached
   - 요금:
     - 노드당 시간당 $0.067 (약 9,000원)
   - 비용 예시:
     - 100GB 캐시 데이터를 `cache.t3.medium` 노드로 1시간 사용
     - 총 비용: $0.067 (약 9,000원)
